Encoding With Quick Base64

Encoding an object into base64 is easy with Quick Base64.
However, there are many ways which you can encode your object. Some of the methods include:

  • Dropping a file on the application.
  • Dropping text onto the application.
  • Loading a text from your system clipboard.
  • Loading a file from your system clipboard.
  • Loading a file using your OS file selector

Dropping a file on the application

To use this method to encode a file, the radiobutton must be set to encode.
Then, you drop a file and the encoded results will be displayed in the next panel.

Dropping a text onto the application to encode

To encode a group of text, just select all the text you want to encode
Then you drag the text from your application and drop it onto Quick Base64.
After dropping the file, the encoded contents will be shown in the next panel.

Loading a file from system clipboard

Quick Base64 allows you to encode a file which is in your system clipboard.
To encode a file, first copy the file to your clipboard.
Then open the application and right-click on the drop area.
A context menu will appear then you select Clipboard (File) option.
The encoded results will appear in the results panel.

Loading a text from system clipboard

You can also encode text which you have copied to your system clipboard.
First copy the text to your clipboard.
Then you right-click the drop area.
A context menu will appear then you select the Clipboard (text) option.

Loading a file using your OS file selector

Also, you can select a file using your operating system file selector.
