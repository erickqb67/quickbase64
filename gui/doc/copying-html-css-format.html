Copying encoded contents in HTML / CSS format

This option is mainly for web developers, it allows you to copy the results in a way which you can insert in your html or css file.
Please note that this option is only available for base64 contents.

Copying contents in HTML format

Also note that, this option only works for images, audios and videos.
When ever you encode a file which belongs to any of these three groups, Quick Base64 will allow you to copy it in HTML format.
This might sound confusing so let me give an example,
If you encode an image of type JPG and you copy it in HTML format, you will get <image src='data:image/jpg;base64,base64contents'>.
Meaning Quick Base64 will generate the coresponding html tag with it's attributes.
Let's try another example and encode a video of type avi.
We will get <video controls src="data:video/avi;base64,base64contents">.
So this way, the application takes care of everything, you just have to paste it into your text editor.

Copying contents in CSS format

Please note, this works for only images.
In CSS, when specifying the background image of an element, you can use base64.
This application makes it easy to get your image into your css file.
For example, if you copying an image in CSS format, you will get a results similar to this.
url(data:image/jpg;base64, base64contents). This enables you to simply copy it inside your CSS file.
